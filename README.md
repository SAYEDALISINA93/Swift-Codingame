# Swift-Codingame

![Swift](https://img.shields.io/badge/Swift-5.0+-orange.svg)
![License](https://img.shields.io/badge/License-MIT-blue.svg)

CodinGame Solutions implemented in Swift Language.

## About

This repository contains solutions to various [CodinGame](https://www.codingame.com/) challenges and puzzles written in Swift. CodinGame is a platform for developers to improve their coding skills through fun and challenging programming puzzles.

## Structure

```
Swift-Codingame/
├── Easy/           # Easy difficulty challenges
├── Medium/         # Medium difficulty challenges
├── Hard/           # Hard difficulty challenges
└── VeryHard/       # Very hard difficulty challenges
```

## Requirements

- Swift 5.0 or later
- Xcode 11.0+ (for macOS)
- Swift compiler (for Linux)

## Running Solutions

Each solution is a standalone Swift file that can be run directly:

```bash
swift <solution-file>.swift
```

Or compile and run:

```bash
swiftc <solution-file>.swift -o output
./output
```
